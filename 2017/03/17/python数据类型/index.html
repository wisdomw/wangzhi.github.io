类型的概念

1、数字类型

4、列表类型

数字类型

python语言包括三种数字类型：
1、整数类型
2、浮点数类型
3、复数类型
三种类型存在一种逐渐“扩展”的关系：
1、整数->浮点数->复数 （整数是浮点数的特例，浮点数是复数的特例）
2、不同的数字类型之间可以进行混合运算，结果为最宽类型，如

不同的数据类型之间可以相互转换，如：

print (“\”大家好\””) #输出——“大家好”

a = “hello woeld”
print (a[1]) #结果是‘e’
print (a[-1]) #结果是‘d’
print (a[1:2]) #结果是‘el’

‘a’+’b’=’ab’
3*’a’=’aaa’

列表

1、列表（list）是有序的元素集合

方法 含义
.append(x) 将元素x添加到列表的最后
.sort() 将列表元素排序
.reverse() 将元素反转
.index(x) 返回第一次出现元素x的索引值
.insert(i,x) 在位置i处插入新元素x
.count(x) 返回x元素在列表中的数量
.remove() 删除列表中第一次出现的x元素
.pop(i) 删除列表中位置i的元素
