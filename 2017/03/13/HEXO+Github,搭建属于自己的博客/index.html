HEXO+Github,搭建属于自己的博客

Mar 13, 2017

今天主要是利用HEXO+Github,搭建属于自己的博客

第一步 安装node
作用：用来生成静态页面的
到Node.js官网下载相应平台的最新版本，一路安装即可。

第二步 安装git
作用：把本地的hexo内容提交到github上去.
安装Xcode就自带有Git
npm install hexo-deployer-git –save

第三步 申请GitHub
作用：是用来做博客的远程创库、域名、服务器之类的

第四步 正式安装Hexo
执行如下命令安装Hexo：
sudo npm install -g hexo

第五步 初始化
终端cd到一个你选定的目录，执行hexo init命令：
hexo init

第六步 安装npm
npm install

第七步 本地启动hexo
hexo s

第八步 关联Github
修改本地hexo生成的文件夹中config.yml文件
deploy:
type: git
repository: https://github.com/XXXX/XXXX.github.io.git
branch: master

第九步 生成静态页面
$ hexo generate 或者：hexo g

第十步 执行配置命令
hexo deploy 或者：hexo d

安装主题：
https://hexo.io/themes/

发布文章／修改主题
1、hexo clean //清除缓存文件 (db.json) 和已生成的静态文件 (public)
2、hexo g //生成缓存和静态文件
3、hexo d //重新部署到服务器
